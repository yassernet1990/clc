# CLC Company Website

A cinematic, bilingual corporate website for an integrated MEP contractor specializing in:

- Fire protection systems
- Fire pumps and pump rooms
- HVAC and ventilation
- Electrical systems
- Plumbing and public health
- BIM coordination, testing and commissioning

## Features

- Full-screen video backgrounds across every major section
- English / Arabic language switch
- Responsive mobile navigation using the native Popover API
- Native HTML dialog for project intake
- View Transitions API enhancement with graceful fallback
- IntersectionObserver video pausing and scroll reveals
- Reduced-motion support and accessible focus states
- Static deployment: no build step required

## Local preview

```bash
python -m http.server 8080
```

Open `http://localhost:8080`.

## Before production launch

1. Replace `projects@clc-company.com` in `script.js` with the official company email.
2. Replace stock video URLs with owned, optimized WebM/MP4 assets if available.
3. Add the official CLC logo, company address, registrations, certifications and real project portfolio.
4. Connect the form to a secure backend or approved form service.
